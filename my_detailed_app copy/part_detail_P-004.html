รายละเอียด Part Code P-004

คุณกำลังดูรายละเอียดสำหรับ: P-004

รหัสพนักงาน:

ข้อมูล Part

ชื่อ Part: Part Code P-004 (สายเคเบิลใยแก้วนำแสง)

รายละเอียด: สายเคเบิลใยแก้วนำแสงแบบ Multi-mode ความยาว 10 เมตร สำหรับการส่งข้อมูลความเร็วสูงในระยะใกล้

ราคา: 2,100 บาท

วิธีการทำงาน

ส่งข้อมูลในรูปของแสงผ่านเส้นใยแก้วขนาดเล็ก แสงจะสะท้อนภายในเส้นใยไปจนถึงปลายทาง โดยมีการสูญเสียน้อยมากเมื่อเทียบกับสายทองแดง

จุดเฝ้าระวัง

  • ตรวจสอบความเสียหายทางกายภาพของสายเคเบิล เช่น รอยหักงอหรือรอยฉีกขาด
  • ตรวจสอบค่า Loss ของสัญญาณแสงที่ปลายทาง
  • หลีกเลี่ยงการดึงหรือบิดสายเคเบิลอย่างรุนแรง
  • ทำความสะอาดหัว Connector เป็นประจำ
กลับไปหน้าเลือก Part Code
